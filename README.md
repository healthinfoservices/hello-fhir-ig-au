# Template FHIR Implementation Guide
This repository contains the source and build files for the Template FHIR Implementation Guide - R5.  This Implementation Guide template has been developed to assist in those getting started with IG Authoring.  It is entirely 'unofficial' but is structured to give a good starting point for a new IG.  Enjoy.
